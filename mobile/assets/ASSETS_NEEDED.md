# Assets Necesarios para RIZE

Por ahora hay placeholders temporales. Para producción, necesitás:

## 1. Icon (1024x1024)
- **Archivo**: `icon.png`
- **Tamaño**: 1024x1024px
- **Formato**: PNG con transparencia
- **Contenido**: Logo RIZE en verde esmeralda (#10B981)

## 2. Splash Screen (1284x2778)
- **Archivo**: `splash.png`
- **Tamaño**: 1284x2778px (iPhone 14 Pro Max)
- **Formato**: PNG
- **Contenido**: Logo RIZE centrado + fondo degradado

## 3. Adaptive Icon Android (1024x1024)
- **Archivo**: `adaptive-icon.png`
- **Tamaño**: 1024x1024px
- **Formato**: PNG con transparencia
- **Contenido**: Logo RIZE (el sistema Android le pondrá fondo)

## 4. Favicon (48x48)
- **Archivo**: `favicon.png`
- **Tamaño**: 48x48px
- **Formato**: PNG
- **Contenido**: Logo RIZE simplificado
